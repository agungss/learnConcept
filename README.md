# Intro
A simple tool to analyze a source code, and count its words/keywords. Duplicates are shown by numbers.

# Usage
java -jar learnConcept.jar <source_code> <exception.txt>

# What is Exception?
In case you are familiar with certain keywords and don't want to include it in the counting, add the keywords there.
It also makes a file result.txt that you can see for reference.
